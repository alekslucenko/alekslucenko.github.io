My Codecademy HTML cheatsheet

By: monkeywarehouse

Tag Name Use
<p> Paragraph Creates a paragraph of text.
<h1> to <h6> Headers 1 through 6 Creates a large header text. Becomes smaller from h1 to h6
<table> Table Creates a table using rows, columns and table headers (<th>) on top. To enter data into a table row, you use the table data (<td>).
